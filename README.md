## 🚀 Featured Projects

### 🎯 [6Recoil](https://github.com/TheX24/6Recoil) (Archived)

- **What it is:** Advanced recoil control tool for Rainbow Six Siege. (Archived due to macro detections)
- **Stack:** Python, tkinter, pynput, pydirectinput

### 🎵 [Lyrprep](https://github.com/TheX24/Lyrprep)

- **What it is:** Web application for formatting and preprocessing song lyrics for [Spicy Lyrics](https://spicylyrics.org).
- **Stack:** HTML5, TypeScript, CSS, Vite, Docker

### 🎮 [DD2RL](https://github.com/TheX24/DD2RL)

- **What it is:** A script to use the **DrunkDeer G75** keyboard as a virtual Xbox controller for analog-style movement.
- **Stack:** Python, low-level input handling

### 🎵 [Spicy Player](https://github.com/TheX24/Spicy-Player)

- **What it is:** Offline Android music player with a port of Spicy Lyrics.
- **Stack:** Kotlin, Jetpack Compose, Media3 ExoPlayer

### 🔋 [Xiaomi-like Battery Bar](https://github.com/TheX24/Xiaomi-like-Battery-Bar)

- **What it is:** A recreation of the battery style found in some Xiaomi devices.

### 🤖 [@Gemini](https://github.com/TheX24/Gemini)

- **What it is:** Google Gemini integrated in a Discord self-bot.
